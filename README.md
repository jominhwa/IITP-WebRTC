# WebRTC
WebRTC를 이용한 1:1 화상회의 웹 만들기

-------
## 환경세팅
```c
Ubuntu 18.04
Node 14.17.3 이상
npm 6.14.14
```

## 기능

+ 닉네임 설정
+ 방 생성
+ 방 리스트
+ 방 인원수 
+ 방 삭제
+ 음소거 & 카메라 on/off
+ 방 나가기
